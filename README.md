# Invoker

A demo project which can load and call rust scripts. 

# Prerequisites

This demo uses rust compiler API. Please switch to nightly.

# Usage 

```bash
cargo run --bin invoker
```

Now the demo only supports numeric.

## Commands

1. load <script> 
2. inspect 
3. call <name> <arg1> <arg2>
4. unload 
5. exit
